# Submission Checklist

## Before Submission

### Format
- [ ] Correct document class / template for venue
- [ ] Within page limit (excluding references unless stated otherwise)
- [ ] Correct font size and margins
- [ ] Anonymous (if double-blind review)

### Content
- [ ] Title is clear and descriptive
- [ ] Abstract is self-contained and within word/character limit
- [ ] All sections complete (no placeholders)
- [ ] All figures and tables referenced in text
- [ ] All claims supported by citations or evidence

### References
- [ ] All citations resolve (no `??` in PDF)
- [ ] Bibliography entries are complete and consistent
- [ ] No orphan references (in bib but not cited)
- [ ] No missing references (cited but not in bib)

### Compilation
- [ ] Compiles without errors
- [ ] No critical warnings
- [ ] PDF renders correctly

### Final
- [ ] Co-authors have reviewed and approved
- [ ] Submission portal account set up
- [ ] Supplementary materials prepared (if applicable)
- [ ] Submission uploaded before deadline
