# Memory Bank

Living documentation system for this paper/proposal project.

## Core Files

| File | Purpose | Update Frequency |
|------|---------|-----------------|
| `projectbrief.md` | Core objectives and requirements | Rarely |
| `productContext.md` | Research problem and proposed approach | When approach changes |
| `techContext.md` | Technical details and methodology | When tech decisions made |
| `systemPatterns.md` | Research patterns and framework | When patterns established |
| `activeContext.md` | Current focus and next steps | Every session |
| `progress.md` | Task completion tracking | Every task completion |
| `phases.md` | Phase coordination hub | Phase transitions |
| `architecturalDecisions.md` | ADR-format decision log | New decisions |

## Reading Order

1. `projectbrief.md` — What are we doing and why?
2. `productContext.md` — What problem are we solving?
3. `activeContext.md` — Where are we now?
4. `progress.md` — What's done and what's next?

## Archive

Completed phases and superseded decisions are moved to `archive/`.
