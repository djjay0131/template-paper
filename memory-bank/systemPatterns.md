# System Patterns

## Documentation Approach

- **Memory-Bank**: Living documentation, updated every session
- **Construction**: Design-first workflow for all content
- **Agents**: Specialized agents for LaTeX, paper type, memory, and review

## Writing Patterns

- Design document before drafting any section
- Sprint-based writing with clear deliverables
- Citation matrix maintained alongside writing
- Compile check after every significant change

## Review Patterns

- Self-review before PR
- Agent review (citations, compilation, content quality)
- Iterate until review-agent approves
